# Weatber-WebApp

A simple and intuitive web application that provides real-time weather updates for any city worldwide. Built with modern web technologies, it delivers accurate forecasts, temperature details, humidity levels, wind speed, and more — all in a clean and responsive interface.

# Features
- Search weather by city name

- Real-time weather data using OpenWeather API (or your chosen API)

- Detailed metrics: temperature, humidity, wind speed, and conditions

- Responsive design for desktop and mobile

- Minimal and user-friendly UI

# Tech Stack
Frontend: HTML, CSS, JavaScript.

API: OpenWeatherMap (or other weather API)

Deployment: GitHub Pages / Netlify / Vercel
